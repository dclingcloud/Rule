# 应用管理需求文档（按“数据筛选/数据展示/操作按钮”）

## 1. 文档说明

- 模块：`应用管理`
- 版本：`v1.1`
- 更新时间：`2026-05-22`
- 目标：明确本模块的**数据筛选规则、数据展示规则、操作按钮规则、L7协议组配置规则与交互规则**，作为产品/开发/测试统一口径。

## 2. 数据筛选

### 2.1 筛选条件定义

- **协议分类筛选**：`IPv4应用` / `IPv6应用` / `混栈应用`。
- **二级类型筛选（Tab）**：
  - 常规：`自定义应用(TCP)`、`自定义应用(UDP)`、`已知应用(TCP)`、`已知应用(UDP)`、`IP应用`、`L7应用`
  - 混栈：`自定义应用(TCP)`、`自定义应用(UDP)`、`IP应用`、`L7应用`
- **表格过滤栏筛选**：
  - 应用名称（模糊匹配）
  - IP（对源IP/目的IP做模糊匹配）
  - 端口（对源端口/目的端口做模糊匹配）
- **顶部上下文下拉**（`实时探针接口/离线任务` + 接口选择）：仅作为上下文展示，当前版本**不参与列表过滤计算**。

### 2.2 筛选交互规则

- 切换协议分类后：
  - 若当前 Tab 不存在于目标分类，自动切到目标分类第一个 Tab。
  - 清空当前批量勾选状态。
  - 关闭浮层菜单（规则优先级菜单、L7协议组优先级菜单）。
- 切换 Tab 后：
  - 清空过滤栏输入与已应用过滤条件。
  - 清空当前批量勾选状态。
  - 关闭浮层菜单（规则优先级菜单、L7协议组优先级菜单）。
- 过滤栏执行方式：
  - 点击“查询”生效；
  - 在过滤输入框按 `Enter` 生效；
  - 点击“重置”清空并恢复默认列表。

### 2.3 筛选结果展示规则

- 列表结果 = 协议分类匹配 AND Tab 匹配 AND 过滤栏条件匹配。
- `L7应用` 下结果按协议组分段展示（见 3.3），过滤仍先于分组生效。

## 3. 数据展示

### 3.1 列表字段与展示规则

- 固定展示列：
  - 序号
  - 应用ID（纯数字）
  - 应用名称
  - 目的IP
  - 目的端口
  - 操作
- 可选列（默认隐藏，可在字段控制中打开）：
  - 源IP
  - 源端口
- 接口列不在主表展示（已移除）。

### 3.2 字段取数与格式规则

- **应用ID**：
  - 展示为数字；
  - 当顶部接口为“所有接口”时，显示“接口-应用ID映射”入口；
  - 其他接口时展示当前接口对应的物理映射ID（悬浮可看完整值）。
- **IP/端口字段**：
  - 超长文本省略显示，悬浮展示完整值。
- **空白行补齐**：
  - 当结果不足一页时，用空行补齐视觉高度，保证表格稳定。

### 3.3 分组展示规则

- **L7应用分组**：
  - 协议组固定：`HTTP`、`DNS`、`SSL`、`Oracle`、`MySQL`、`PostgreSQL`；
  - 仅展示有数据的协议组；
  - 分组标题行形如：`协议组：HTTP`；
  - 协议组之间支持“整组排序”：
    - 支持直接拖拽协议组；
    - 支持点击协议组左侧拖拽图标后通过菜单排序（置顶/上移/下移/置底）。
  - 组内应用支持优先级排序：
    - 支持组内拖拽排序；
    - 支持组内优先级菜单排序（置顶/上移/下移/置底）；
    - 排序范围限制在当前协议组内，不可跨组。
  - **协议组级配置入口**：
    - 当协议组为 `HTTP`、`DNS`、`SSL`、`Oracle`、`MySQL`、`PostgreSQL` 时，分组标题行右侧展示 **「设置」** 按钮；
    - 按钮仅在对应协议组存在数据（出现分组标题行）时可见；
    - 点击后打开该协议组专属配置弹窗（见第 5 章）。

### 3.4 特殊数据展示规则

- **已知应用（TCP/UDP）**：
  - 支持“配置”只读回显；
  - 不支持编辑修改。
- **IP默认应用**（`ICMP`、`multicast_test`、`unicast_test`、`ip_other`）：
  - 在 `IP应用` 下作为默认规则展示；
  - 支持“配置”只读回显；
  - 不支持编辑修改。

## 4. 操作按钮

### 4.1 页面级按钮显示规则

- **新增应用**：
  - 位置：Tab/过滤同一操作行；
  - 在 `已知应用(TCP/UDP)` 下禁用；
  - 在其他 Tab 启用，新增内容归属当前协议分类 + 当前Tab。
- **表格过滤**：始终显示，可展开/收起过滤栏。
- **批量高级配置**：
  - 在 `已知应用(TCP/UDP)` 下禁用；
  - 其他 Tab 启用；
  - 需先勾选至少 1 条应用。
- **导入配置 / 导出配置 / 下载模板 / 一键清空**：
  - 常驻显示；
  - 一键清空作用于“当前协议分类 + 当前Tab”。

### 4.2 行级按钮显示规则（操作列）

- **自定义应用/IP应用/L7应用（可编辑项）**：
  - 显示：复制、编辑、高级配置、删除、启用开关。
- **已知应用(TCP/UDP)**：
  - 显示：配置（编辑 icon，点击只读回显）。
- **IP默认应用**：
  - 显示：配置（编辑 icon，点击只读回显）。
- 操作列表头统一文案：`操作`。

### 4.3 按钮交互规则

- **新增**：
  - 打开可编辑弹窗；
  - 必填校验通过后保存；
  - L7应用新增时必须选择协议组。
- **编辑**：
  - 打开可编辑弹窗并回显；
  - 规则定义为全局统一；
  - 编辑应用时接口只允许“新增”，不允许删除已有接口。
- **复制**：
  - 打开可编辑弹窗并回显；
  - 名称默认 `副本-原名称`；
  - 保存后生成新记录。
- **高级配置（单条）**：
  - 独立入口打开高级配置弹窗；
  - 非只读项可保存；
  - 高级配置支持批量入口统一覆盖。
  - 弹窗不再提供“当前接口配置”切换区；
  - 高级配置字段包含：连接超时阈值、应用性能时延区间、存包长度、底层参数调优（SRv6解析、连接中间状态处理、规则平衡等）。
- **配置（只读回显）**：
  - 入口用于已知应用/IP默认应用；
  - 弹窗仅展示，不可修改；
  - 无底部提交按钮，仅右上角关闭。
- **启用开关（操作列最右侧）**：
  - 展示当前应用状态（启用/停用）；
  - 点击后先弹状态切换确认弹窗；
  - 确认后切换，取消则保持原状态不变。
- **删除**：
  - 点击后先弹二次确认；二次确认弹窗中需要提示“删除当前应用后，相关功能（如应用拓扑、应用组等）中的应用将会被取消关联，确认删除？”
  - 确认后执行删除；
  - 取消则不变更数据。
  - 应用删除后需要处理的模块：
    - 应用拓扑节点（置为空节点）
    - 应用组定义（原有N个应用，现有N-1个应用；若应用组中所有应用被删除，那么应用组也被删除）
    - 工作台应用相关视图（原有N个应用，现有N-1个应用；若视图中所有应用被删除，那么视图也被删除）
    - 工作台-长周期分析报表（原有N个应用，现有N-1个应用；若报表中所有应用被删除，那么报表也被删除）
- **批量高级配置**：
  - 对勾选规则统一应用高级参数；
  - 保存后覆盖目标规则高级参数配置。

### 4.4 勾选与批量规则

- 表头全选仅作用于当前筛选结果集。
- 切换协议分类或Tab时自动清空勾选。
- 删除后自动清理对应勾选状态。

## 5. L7协议组配置

> 本章说明 `L7应用` Tab 下六个协议组（`HTTP`、`DNS`、`SSL`、`Oracle`、`MySQL`、`PostgreSQL`）的「设置」入口、交互规则与配置项明细。保存维度均为 **当前协议分类 + 当前协议组**。

### 5.1 入口与展示规则

- **入口位置**：协议组标题行右侧 **「设置」** 按钮；仅当该协议组存在数据（出现分组标题行）时可见。
- **配置维度**：协议组级配置，作用于当前协议分类下的对应协议组，不作用于单条应用。
- **独立保存**：不同协议分类（`IPv4应用` / `IPv6应用` / `混栈应用`）下的同一协议组配置相互独立保存与回显。
- **弹窗样式**：左对齐表单布局；启用/不启用类选项以下拉框展示；数据源调度以复选框多选展示。
- **弹窗标题**：分别为 `DNS配置`、`HTTP配置`、`SSL配置`、`MySQL配置`、`Oracle配置`、`PostgreSQL配置`。

### 5.2 交互规则

- **打开**：点击「设置」后打开对应协议组配置弹窗；回显当前协议分类下已保存配置；若无历史配置，按 5.4～5.9 默认值回显。
- **保存**：点击弹窗底部 **「修改」** 保存；保存后关闭弹窗，不影响列表中应用条目的增删改。
- **关闭**：点击右上角关闭或遮罩层关闭；未点击「修改」则本次编辑不生效。
- **边界**：与应用级「编辑 / 高级配置 / 复制 / 删除」相互独立；不替代单条应用的基础定义与高级定义。

### 5.3 通用控件规则

- **启用/不启用类字段**：下拉选择，选项为 `启用` / `不启用`。
- **数据源调度**：复选框多选，选项为 `ClickHouse`、`MongoDB`，至少可选 0 项、最多可选 2 项。
- **保存按钮**：弹窗底部 **「修改」**；保存后关闭弹窗。

## 6. 验收要点

- 数据筛选：筛选条件、触发方式、清空规则、结果计算规则明确且一致。
- 数据展示：字段显示、分组显示、空行补齐、特殊数据（已知/IP默认）展示规则明确。
- 操作按钮：按钮显示条件、禁用条件、交互动作、只读与可编辑边界明确。
- L7协议组配置：六个协议组「设置」入口、交互规则、配置项与默认值（第 5 章）明确且一致。
